## Introduction

This is a simple chatroom just for testing Laravel 11 & Livewire, wanna tried Laravel Reverb but Livewire is good enough.

## Requirements

1. PHP 8.3
2. Node.js 18^

## Installation

```sh
composer install
php artisan migrate

npm install
npm run build
```

## Start Server

```sh
Terminal 1: php artisan serve
Terminal 2: npm run dev
```

go to [http://127.0.0.1:8000]
